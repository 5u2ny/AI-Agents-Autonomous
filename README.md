# AI Agents Autonomous

Practical experiments in autonomous agent workflows: planning, tool use, routing, memory/context handling, and human-readable outputs.


## What This Demonstrates

- **Task decomposition** — splitting broad goals into smaller executable steps.
- **Tool-use patterns** — treating APIs, files, and local scripts as controlled capabilities.
- **Workflow orchestration** — sequencing agent actions instead of relying on one-shot prompts.
- **Memory and context** — keeping useful state without turning the system into an opaque black box.
- **Human-in-the-loop control** — designing agent outputs that support review, approval, or next action.
